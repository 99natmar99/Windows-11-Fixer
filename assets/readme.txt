This program requires ".net 5.0 Desktop Runtime" in order to run.

If you are running the x64 version of Windows, run "Install net5.0 (x64).bat" to install the runtime.

If you are running the x86 version, you can download the runtime from Microsoft's website.

Attempting to run the program without .net 5.0 installed will result in being prompted to download the runtime from Microsoft's website.
